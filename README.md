# 古文岛助手

[古文岛](https://www.guwendao.net) 油猴脚本。

## 功能

- **登录绕过**：自动注入 cookie 跳过登录校验，无需注册即可使用全站功能
- **无限朗读**：解除朗读次数付费限制
- **PDF 免验证下载**：跳过验证直接导出 PDF，支持水印开关
- **音频下载**：直接下载朗读音频 MP3

## 安装

Greasy Fork 搜索"古文岛助手"一键安装，或从 [Releases](https://github.com/wankong/guwendao-render/releases) 下载 `guwendao-speaker.user.js`。

## 开发

```bash
pnpm install
pnpm dev      # 开发
pnpm build    # 构建 → dist/guwendao-speaker.user.js
```
