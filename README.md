# 古文岛助手

![Greasy Fork Version](https://img.shields.io/greasyfork/v/586065) ![Greasy Fork Downloads](https://img.shields.io/greasyfork/dt/586065) ![Greasy Fork Rating](https://img.shields.io/greasyfork/rating-count/586065)


[古文岛](https://www.guwendao.net) 油猴脚本。

## 功能

- **登录绕过**：自动注入 cookie 跳过登录校验，无需注册即可使用全站功能
- **无限朗读**：解除朗读次数付费限制
- **PDF 免验证下载**：跳过验证直接导出 PDF，支持水印开关
- **音频下载**：直接下载朗读音频 MP3


## 安装

Greasy Fork 搜索"[古文岛助手](https://greasyfork.org/zh-CN/scripts/586065-%E5%8F%A4%E6%96%87%E5%B2%9B%E5%8A%A9%E6%89%8B)"一键安装，或从 [Releases](https://github.com/wan-kong/guwendao-unlock/releases) 下载 `guwendao-unlock.user.js`。

## 开发

```bash
pnpm install
pnpm dev      # 开发
pnpm build    # 构建 → dist/guwendao-unlock.user.js
```
